# 002-npm源切换——nrm

<motto></motto>

| 描述           | 指令                                 |
| -------------- | ------------------------------------ |
| 查看可选源     | nrm ls                               |
| 测试源响应时间 | nrm test taobao                      |
| 切换源         | nrm use taobao                       |
| 增加定制源     | nrm add selfnpm http://127.0.0.1:666 |
| 删除源         | nrm del selfnpm                      |
